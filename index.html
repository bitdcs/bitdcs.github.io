About Coding Society

The best way to excel in competitive coding is to, well, compete. The Coding Club orgainzes regular contests on popular coding platforms in order to inculcate the coding culture in students. In addition, We organize lectures on Programming Languages and algorithms & data structures in order to better place you among the global competition.
